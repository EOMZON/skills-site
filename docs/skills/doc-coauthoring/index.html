写作与文档

Doc Co-Authoring

结构化文档共创 workflow，适合提案、Spec、PRD、决策文档与长文写作。

GitHub First

完整公开说明放在 GitHub，不放在站内长期镜像。

这个页面只保留快速判断和调用契约。真正的公开文档、更新历史和 star 入口都应该回到 GitHub。

Read on GitHub Browse Folder Open Repo

Invoke

$doc-coauthoring

Inputs

doc_goal 文档类型、目标和主要读者 (required)
source_context 背景信息、模板、已有草稿、会议记录或相关链接

Returns

文档结构与章节 scaffold
逐段共创与修订建议
读者视角测试与缺口清单

Use When

要写结构化文档，但上下文还很散
希望边梳理边成稿，而不是一次性吐整篇
想在发出去前做一次读者视角检查

Avoid When

只需要润色一小段文本
还没有明确的文档目标或受众
