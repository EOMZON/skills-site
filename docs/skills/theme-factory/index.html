Theme Factory

为 HTML、文档、幻灯或报告选择并应用成体系的视觉主题，而不是零散改样式。

GitHub First

完整公开说明放在 GitHub，不放在站内长期镜像。

这个页面只保留快速判断和调用契约。真正的公开文档、更新历史，以及 star / follow 行为都应该回到 GitHub。

Read on GitHub Browse Folder Open Repo Follow @EOMZON

Invoke

$theme-factory

Inputs

artifact_type 要应用主题的产物类型，例如 HTML、文档或幻灯 (required)
style_goal 希望呈现的气质、受众和使用场景

Returns

主题 shortlist 或自定义方向
颜色与字体 tokens
应用到产物时的统一样式建议

Use When

已有文档或页面，需要统一视觉主题
要快速比较不同风格方向
需要把主题 tokens 落到实际产物里

Avoid When

还没确定内容结构或信息架构
只想讨论抽象审美而不落到产物上
