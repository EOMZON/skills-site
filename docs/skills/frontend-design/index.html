Frontend Design

Create distinctive, production-grade frontend interfaces with clear visual direction.

Use When

  • 需要做网站、落地页、Dashboard 或内容型页面
  • 需要把设计方向和前端实现一起推进

Input

  • goal：你要做的页面或界面
  • constraints：风格、技术栈、设备、交付要求

Returns

  • 设计方向
  • 前端实现
  • 体验和验收建议

Notes

  • 公开版保留调用契约和设计原则
  • 上游私有源里的本地模板路径和团队偏好已移除

Invoke

$frontend-design

Inputs

goal 要做什么页面、组件或交互 (required)
constraints 技术栈、风格、设备适配、交付范围等约束

Returns

设计方向
页面或组件实现
验收与优化建议

Use When

需要设计并实现网页或前端界面
希望页面有清晰的风格方向而不是通用模板感

Avoid When

只是讨论抽象设计概念
不需要任何实际前端交付
