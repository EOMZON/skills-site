Zon Minimal Editorial

把内容型页面做成可读、可回看、可归档的出版物式体验，而不是默认落回 SaaS 模板。

Use When

  • 需要为网站、报告、作品集、博客或档案页建立统一的 editorial house style
  • 需要黑白克制、内容优先的视觉方向
  • 需要减少卡片墙和装饰性 UI，让层级、节奏和排版承担主要表达

Input

  • artifact_type：要处理的页面或产物类型
  • content_shape：现有结构、页面问题或目标阅读体验

Returns

  • 适配判断和页面原型方向
  • typography / spacing / hierarchy 约束
  • 可复用的 house style 执行建议

Notes

  • 公开版保留风格原则和落地规则
  • 私有模板路径与项目内默认资源已抽象掉

Invoke

$zon-minimal-editorial

Inputs

artifact_type 要处理的页面或产物类型，例如网站、作品集、报告页、博客或档案页 (required)
content_shape 当前信息结构、已有页面骨架、视觉问题或期望的阅读体验

Returns

editorial fit 判断与页面原型方向
层级、排版和视觉约束
可复用的 house style 执行建议

Use When

要把网站、报告、作品集或档案页做成出版物式阅读体验
需要统一黑白克制的 house style
想减少 SaaS 模板感，让结构和排版承担主要视觉工作

Avoid When

页面主要任务是高频交易或彩色状态监控
需要强烈娱乐感、插画感或营销活动页气质
