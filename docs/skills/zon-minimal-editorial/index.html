Zon Minimal Editorial

把内容型网站和页面收口为克制、黑白、内容优先的 editorial house style。

GitHub First

完整公开说明放在 GitHub，不放在站内长期镜像。

这个页面只保留快速判断和调用契约。真正的公开文档、更新历史，以及 star / follow 行为都应该回到 GitHub。

Read on GitHub Browse Folder Open Repo Follow @EOMZON

Invoke

$zon-minimal-editorial

Inputs

artifact_type 要处理的页面或产物类型，例如网站、作品集、报告页、博客或档案页 (required)
content_shape 当前信息结构、已有页面骨架、视觉问题或期望的阅读体验

Returns

editorial fit 判断与页面原型方向
层级、排版和视觉约束
可复用的 house style 执行建议

Use When

要把网站、报告、作品集或档案页做成出版物式阅读体验
需要统一黑白克制的 house style
想减少 SaaS 模板感，让结构和排版承担主要视觉工作

Avoid When

页面主要任务是高频交易或彩色状态监控
需要强烈娱乐感、插画感或营销活动页气质
