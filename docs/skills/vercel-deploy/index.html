Vercel Deploy

将现有工程发布到 Vercel，并处理分析与域名配置。

Invoke

$vercel-deploy

Inputs

project_path 待发布项目路径或仓库根目录 (required)
domain 可选自定义域名或子域名

Returns

部署结果
线上访问地址
必要的后续配置建议

Use When

需要把工程发布到 Vercel
需要配置域名、分析或上线检查

Avoid When

项目还没有可发布内容
只是需要本地预览而不是上线
