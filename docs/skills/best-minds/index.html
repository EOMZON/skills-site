Best Minds

默认思维 skill：适用于调研、思路梳理、路线规划与决策。

Use When

  • 你面对的是复杂问题，不知道该从哪里拆解
  • 你想借用“世界上谁最懂这个问题”的视角来组织研究

Input

  • question：核心问题
  • scope：范围、对象、时间、限制

Returns

  • 结论摘要
  • 发现与证据
  • 建议与下一步验证

Notes

  • 公开版只保留方法论与调用契约
  • 具体私有工作流、内部 board/deploy 逻辑应留在私有作者源

Invoke

$best-minds

Inputs

question 要研究或回答的核心问题 (required)
scope 范围、对象、时间、地域、排除项

Returns

结论摘要
关键发现与证据
建议与下一步验证

Use When

需要快速理解一个复杂问题
需要从顶级专家视角组织分析和建议

Avoid When

只需要简单事实而不需要分析
没有任何问题边界和目标
