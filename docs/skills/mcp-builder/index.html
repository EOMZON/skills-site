MCP Builder

用于把外部 API 或系统组织成高质量 MCP server，而不是只写一组零散工具。

Use When

  • 需要为 agent 暴露一套稳定的工具接口
  • 需要决定用 TypeScript 还是 Python 来实现 MCP server
  • 需要补协议理解、工具设计、测试和 evaluation

Input

  • target_service：要接入的 API、数据源或系统
  • stack_constraints：语言偏好、鉴权方式、运行环境

Returns

  • MCP server 的实现路线
  • 更适合 agent 发现和调用的 tool 设计建议
  • 测试、评估和质量检查清单

Notes

  • 公开版保留协议与工程方法
  • 私有密钥、账号配置和组织内部 API 细节需要由使用者自行补充

Invoke

$mcp-builder

Inputs

target_service 要接入的 API、服务或数据源 (required)
stack_constraints 语言偏好、运行环境、鉴权限制或部署方式

Returns

MCP server 设计原则与实现路线
tool schema 与错误处理建议
测试和 evaluation 清单

Use When

要把外部 API 或服务接入 LLM 或 agent
要设计一组可发现、可组合的 MCP tools
要给 MCP server 补测试和评估

Avoid When

只是调用现成 MCP server，不需要自己实现
问题和协议设计或 API 集成无关
