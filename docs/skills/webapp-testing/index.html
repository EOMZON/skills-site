Webapp Testing

用 Playwright 验证本地 Web 应用行为，适合交互验收、回归检查、截图取证与 UI 调试。

GitHub First

完整公开说明放在 GitHub，不放在站内长期镜像。

Open SKILL.md Open Folder Open Repo

Invoke

$webapp-testing

Inputs

app_url_or_path 待验证的本地地址、页面入口或运行方式 (required)
verification_goal 要检查的关键流程、问题现象或验收标准

Returns

浏览器内的验证结论
截图、日志或复现证据
可复跑的自动化步骤

Use When

需要验证本地网页或前端交互
需要复现 bug、截图取证或查看浏览器日志
需要在真实浏览器里走一遍关键流程

Avoid When

只是阅读静态源码而不需要运行页面
不是 Web 界面相关问题
